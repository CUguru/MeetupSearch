Meetly Web Application
Meetly is a web application that utilizes the meetup API to search for meetup groups.

Technologies used
Meetly uses the following technologies:

- HTML5
- CSS3
- Flexbox
- Meetup API
- jQuery
- Google Fonts
- Font Awesome
- Installation

----To get the application running on your system:----

- download and open the compresser folder
- Using the browser
- Inside the folder, locate the form.html file
- Open up your browser of choice and drag the form.html file into the browser
- Once the file is in the browser, scroll to the input box and enter the group
  you would like to search for.
- If no valid input is entered, an error is thrown telling you that you need to fill
  in the field
- You can choose to enter a zip-code or not, either way the search will run
  without the zip-code.
- The results without the zip-code will use the location based on the
  default API key.
